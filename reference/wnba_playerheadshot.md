# **Get WNBA Stats API Player Head-shot**

**Get WNBA Stats API Player Head-shot**

**Get WNBA Stats API Player Head-shot**

## Usage

``` r
wnba_playerheadshot(player_id = "1628932", ...)
```

## Arguments

- player_id:

  Player ID

- ...:

  Additional arguments passed to an underlying function like httr.

## Value

Returns a url of the png for the player_id selected

## Details

     wnba_playerheadshot(player_id = '1628932')
