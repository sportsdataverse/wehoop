# `espn_wbb_scoreboard`

Get women's college basketball schedule for a specific year from ESPN's API


## Description

Get women's college basketball schedule for a specific year from ESPN's API


## Usage

```r
espn_wbb_scoreboard(season)
```


## Arguments

Argument      |Description

`season`     |     Either numeric or character


## Value

Returns a tibble
