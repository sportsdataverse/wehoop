# `espn_wnba_player_box`

## Description

Get ESPN's WNBA player box data


## Usage

```r
espn_wnba_player_box(game_id)
```


## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID


## Examples

```r
espn_wnba_player_box(game_id = 401244185)
```
