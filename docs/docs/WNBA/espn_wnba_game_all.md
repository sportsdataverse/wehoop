# `espn_wnba_game_all`

## Description

Get ESPN's WNBA game data (play-by-play, team and player box)

## Usage

```r
espn_wnba_game_all(game_id)
```

## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_wnba_game_all(game_id = 401244185)
```
