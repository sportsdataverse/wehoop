# `espn_wnba_team_box`

## Description

Get ESPN's WNBA team box data

## Usage

```r
espn_wnba_team_box(game_id)
```

## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID


## Examples

```r
espn_wnba_team_box(game_id = 401244185)
```
