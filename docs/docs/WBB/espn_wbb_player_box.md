# `espn_wbb_player_box`

## Description

Get ESPN women's college basketball player box


## Usage

```r
espn_wbb_player_box(game_id)
```

## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_wbb_player_box(game_id = 401276115)
```
