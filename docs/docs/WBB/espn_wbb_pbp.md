# `espn_wbb_pbp`

## Description

Get ESPN women's college basketball play by play data


## Usage

```r
espn_wbb_pbp(game_id)
```

## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID


## Examples

```r
espn_wbb_pbp(game_id = 401276115)
```
