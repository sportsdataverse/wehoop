# `espn_wbb_game_all`

## Description

Get ESPN women's college basketball game data (play-by-play, team and player box)

## Usage

```r
espn_wbb_game_all(game_id)
```

## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_wbb_game_all(game_id = 401276115)
```
