# `espn_wbb_standings`

## Description

Get ESPN women's college basketball standings


## Usage

```r
espn_wbb_standings(year)
```


## Arguments

Argument      |Description
------------- |----------------
`year`     |     Either numeric or character (YYYY)


## Examples

```r
espn_wbb_standings(2021)
```
