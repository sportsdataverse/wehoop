# `espn_wbb_team_box`

## Description

Get ESPN women's college basketball team box data


## Usage

```r
espn_wbb_team_box(game_id)
```


## Arguments

Argument      |Description
------------- |----------------
`game_id`     |     Game ID

## Examples

```r
espn_wbb_team_box(game_id = 401276115)
```
